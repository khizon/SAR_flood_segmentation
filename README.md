# SAR_flood_segmentation

## Installation
- Python version: 3.10
- CUDA version: 12.4
- Install pytorch (specific version)
    pip install torch==2.6.0 torchvision==0.21.0 --index-url https://download.pytorch.org/whl/cu124
    pip install torch==2.6.0 torchvision==0.21.0 --index-url https://download.pytorch.org/whl/cu130 (depending on CUDA version)
- Install the rest of the requirements file
    pip install -r requirements.txt
